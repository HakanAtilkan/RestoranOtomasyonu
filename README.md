## Restoran Otomasyonu

Bu proje, bir restoran otomasyon sisteminin tam yığın (fullstack) örneğidir.

- **Backend**: Node.js (Express) REST API
- **Frontend**: React ile SPA arayüz

ER diyagramındaki temel tablolar:

- **ROLLER, KULLANICILAR, ISLEMLER, ROL_ISLEM**
- **MASALAR, SIPARISLER, SIPARIS_DETAY, URUNLER**
- **ODEMELER, ODEME_DETAY**
- **HAMMADDELER, RECETELER, STOK_HAREKETLERI, TEDARIKCILER**

Proje iki ana klasörden oluşur:

- `backend/` — API, veritabanı modelleri ve iş kuralları
- `frontend/` — React ile yazılmış web arayüzü

Kurulum ve çalışma adımları, ilgili klasörlerin içindeki `README.md` dosyalarında anlatılmaktadır.
